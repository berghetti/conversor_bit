conv
===

pequeno programa para conversão de unidades computacionais, bit, byte, Mb, MB...

### [install]
    $ make
  
### [usage]
    use: conv <valor> <opção>
    <valor>
    Ex: 64b
    Ex: 10Mb
    Ex: 10.2MB

    <opção>
    -v    [não utiliza notação cientifica]
    -h    [exibe essa mensagem]

    <exemplo>
    ./conv 100Kb -v
    ./conv 20MiB
